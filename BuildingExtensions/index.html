Building Extensions - Overview

These pages will guide you to create your own extension without touching the framework code. This is possible by dynamically fetching extending content from all assemblies and even enables user extensions.
If you want/need to create modify the framework for your extension, you can refer to the Framework Overview to get a glimpse on how each framework part works.


Covered Topics


- Custom Nodes - Implementation of custom Nodes

- Custom ConnectionTypes - Defining custom ConnectinTypes in order to pass custom types with customized visualization

- Adding NodeCanvas Types - Implementation of custom NodeCanvas Types for customized situations

- Adding Input Controls - Addition of Input Controls to the dynamic Input System with Attribbutes

- Building an Editor - Building of a custom Editor Window or other user of the Node Editor Framework

- Event & Callbacks - Making use of the framework events and callbacks
